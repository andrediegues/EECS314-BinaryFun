Learn About Integers & Floating Points

Integers are generally represented in MIPS as 32 binary digits (“bits”). 32-bit numbers can also be represented as eight hexadecimal characters. If all bits are used for the value of a number, it can only represent non-negative numbers (which means it’s an unsigned value). Unsigned 32-bit integers can represent numbers in the range [0, 232–1]. Signed integers, on the other hand, considers the largest-value bit to be negative. These can represent numbers in the range [-231, 231-1]. The equation and table below demonstrates a signed 4-bit number.

Positive signed numbers are identical to positive unsigned integers. Signed numbers can be converted from positive to negative by inverting each bit (changing zeros to ones and ones to zeros) and then adding 1 to the value. Conversion from negative to positive is the same in reverse.
