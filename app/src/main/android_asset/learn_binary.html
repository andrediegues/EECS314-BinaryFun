Learn Binary & Hexadecimal

Binary and hexadecimal are number systems, just like decimal. Decimal is base ten, which means that it uses ten characters (0 through 9) and each digit is worth ten times the one to the right (e.g. “10” is ten times as much as “1”). Binary is base two: is uses two characters (0 and 1) and each digit is worth twice as much as the one to the right (e.g. “10” is double the value of “1”). Hexadecimal is base sixteen: it uses sixteen characters (0-9 and A-F, representing numbers 0 through 15) and each digit is worth sixteen times the one to the right (e.g. “10” is sixteen times “1”).

Binary is useful for computers because transistors have two states, which can represent “1” (on) and “0” (off). A series of transistors can hold longer binary numbers, which can represent data. Hexadecimal is a useful way for people to represent binary, because it is easy to convert between base 2 and 16.
